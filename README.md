# Scorleaf 🌿

Official website for **Scorleaf** — an Omani student startup developing a natural licorice root extract to relieve the effects of scorpion bites.

## About

Scorleaf is an innovative plant-based first-aid solution backed by scientific research and traditional Omani heritage. The project aligns with **Oman Vision 2040** and has received recognition from government ministries, academic institutions, and national media.

## Features

- **Responsive PWA** — installable as a standalone app on mobile and desktop
- **AI Symptom Checker** — client-side scoring tool to assess scorpion sting severity (mild / moderate / severe) with instant recommendations
- **AI Chatbot** — Arabic keyword-matching assistant that answers questions about Scorleaf, the team, and the product — no API key required, works fully offline
- **Podcast Embed** — embedded Hala FM interview featuring the Scorleaf story
- **RTL Arabic UI** — fully right-to-left layout using Tajawal & Nunito fonts

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Progressive Web App (PWA) with Service Worker
- Font Awesome icons
- SoundCloud embed (public iframe, no API)
- Google Fonts (Tajawal, Nunito, Quicksand)
