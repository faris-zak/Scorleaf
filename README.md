# Scorleaf 🌿

Official website for **Scorleaf** — an Omani student startup developing a natural licorice root extract to relieve the effects of scorpion bites.


## About

Scorleaf is an innovative plant-based first-aid solution backed by scientific research and traditional Omani heritage. The project aligns with **Oman Vision 2040** and has received recognition from government ministries, academic institutions, and national media.
